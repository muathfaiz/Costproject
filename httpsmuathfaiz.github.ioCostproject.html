Estimator Checklist
  • ✓ Convert all dimensions — inches to feet, cubic feet to cubic yards
  • ✓ Add correct waste factor (5% footings, 8% slabs; 1–10% rebar)
  • ✓ Separate items by type (footings, walls, slabs, bar size, location)
  • ✓ Apply labor productivity in LH/CY for concrete, LH/ton for rebar
  • ✓ Replace assumed unit prices with actual supplier quotes before final bid
  • ✓ Keep all assumptions visible for reviewer to update quickly
  • ✓ Include equipment, forms, crane, and erection as separate line items if required
